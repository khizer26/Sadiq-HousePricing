This code provides you with the house price prediction model using Machine Learning for a given dataset. The code is trained in 6 different models; Ordinary Least Squares (OLS), Ridge, Lasso, Bayseian, Elastic Net and Random Forest Regression, It was found out that Random Forest Regression is the best model with least absolute error.

For this code to run you need to install the following packages:
1) scikit-learn: pip install -U scikit-learn
2) pandas: pip install pandas
3) numpy: pip install numpy
4) matplotlib.pyplot: pip install matplotlib
5) seaborn: pip install seaborn
